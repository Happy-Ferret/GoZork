### Linux, MacOS, Windows

```bash
$ go get github.com/XenonLab-Studio/GoZork
```

<br>

**Example $GOPATH for Linux:**

Open a terminal, and navigate to your $GOPATH directory.

```bash
/home/user/go/src/github.com
```

<br>

**Run project**

```bash
$ cd GoZork
$ go run main.go
$ ./GoZork # to start the game!
```

<br>

**Build project**

```bash
$ go build
$ ./GoZork # to start the game!
```
